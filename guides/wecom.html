企业微信接入指南

通过 SudoClaw 的企业微信渠道配置，将 SudoClaw 接入企业微信，随时随地远程操控完成任务

平台接入指南

个人微信接入指南 企业微信接入指南 飞书接入指南 钉钉接入指南

前提条件

  • 已在电脑上安装并登录 SudoClaw
  • 拥有一个可创建智能机器人的企业微信账号

SudoClaw 通过 WebSocket 长连接方式接入企业微信，无需配置 Webhook URL，只需填写 Bot ID 和 Secret 即可完成绑定。

步骤一：创建企业微信智能机器人

1）选择创建入口

可根据您的账号角色选择不同入口：

管理员

打开浏览器，访问企业微信管理后台，使用管理员账号登录后，依次点击「安全与管理」→「管理工具」→「智能机器人」→「创建机器人」

普通成员

打开企业微信客户端，进入「工作台」→「智能机器人应用」→「创建机器人」

2）进入 API 模式创建流程

进入机器人创建页面后，按以下方式切换到 API 模式：

管理员

在创建页面直接选择 「API 模式创建」

普通成员

如果先进入 AI 自动生成页面，请点击左下角的 「手动创建」，再选择 「API 模式创建」

进入 API 模式页面后，填写机器人基本信息并保存：

  1. 1

    填写机器人名称，建议填写如「SudoClaw 助手」

  2. 2

    设置可见范围，选择需要使用机器人的成员、部门或标签

  3. 3

    点击页面底部的 「保存」

步骤二：获取 Bot ID 和 Secret

保存基本信息后，在右侧「API 配置」区域选择「使用长连接」：

  1. 1

    复制 Bot ID

  2. 2

    点击「点击获取」获取 Secret，并妥善保存

长连接模式不需要再填写 URL、Token 或 Encoding-AESKey。请务必保存好 Bot ID 和 Secret。

步骤三：在 SudoClaw 中完成绑定

  1. 1

    打开 SudoClaw

  2. 2

    进入 远程连接 → 渠道配置

  3. 3

    在集成列表中找到 企业微信，并点击

  4. 4

    填入刚才复制的 Bot ID 和 Secret

  5. 5

    点击 「测试并连接」 完成绑定

步骤四：开始使用

配置完成后，您就可以在企业微信中直接与机器人对话了。

  1. 1

    在企业微信通讯录的「企业创建的」分组下找到刚刚创建的机器人

  2. 2

    点击「发消息」即可开始下发任务

  3. 3

    发送一条简单消息（例如「你好」）进行联通测试

如果配置正确，SudoClaw 会接收到消息并在企业微信中返回回复。您也可以在 企业微信渠道配置中选择使用的 Agent（默认为 SudoClaw）。

常见问题

机器人没有响应怎么办？

  1. 检查 SudoClaw 状态：确保电脑上的 SudoClaw 正在运行，且渠道配置服务已开启
  2. 核对接入方式：确认企业微信侧选择的是「使用长连接」
  3. 检查凭据：核对 Bot ID 和 Secret 是否正确，避免带入多余空格
  4. 检查网络连接：确保电脑能够正常访问网络

测试并连接失败怎么办？

  1. 确认企业微信侧已选择「使用长连接」
  2. 重新复制 Bot ID 和 Secret，避免带入多余空格
  3. 如 Secret 已失效，可在企业微信后台重新获取后再次尝试

在企业微信中找不到机器人怎么办？

  1. 确认机器人的可见范围包含您的账号
  2. 在企业微信通讯录的「企业创建的」分组下查找
  3. 尝试刷新企业微信客户端

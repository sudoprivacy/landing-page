文档

了解 SudoClaw 平台的核心概念、快速上手指南与产品架构

快速开始

  1. 1

    申请访问

    发邮件至 business@sudoprivacy.com，我们将在 1 个工作日内回复。

  2. 2

    部署环境

    支持私有化部署（Docker / K8s）和 SaaS 托管两种模式。

  3. 3

    连接模型

    支持 OpenAI、Anthropic、通义千问、DeepSeek、文心一言等主流接口。

  4. 4

    配置 Worker

    从预置 Worker 库中选择，或通过 SDK 开发自定义 Worker。

  5. 5

    开始使用

    通过 Copilot 下达任务，Worker 自动执行并返回结果。

核心概念

🧭

Copilot

意图理解与任务编排层。接收用户自然语言指令，拆解为结构化任务树，调度 Worker 执行。

⚙️

Worker

技能执行单元。每个 Worker 专注一个能力域（数据采集、文档生成、API 调用、系统操作等），可并行运行。

🔀

任务编排

Copilot 根据任务依赖关系自动决定串行或并行执行策略，最大化效率。

🔒

私有部署

全栈私有化，模型推理、数据存储、任务执行全部在企业内网运行，数据不出域。

生成 Landing Page 所需的全部视觉素材（SVG 为主），让页面从"文字墙"变成有视觉层次、专业感的品牌网站。

## Logo 文件

已有素材（在 `assets/images/` 目录下）：
- `logo.svg` — 黑底 + amber 立方体图标（512×512）

## 素材清单

### 1. Nav Logo 组合（文字 + 图标）

产出一个适合 Nav 的横排组合：
```
[立方体图标 32×32] SudoClaw
```
- 图标：从 logo.svg 提取路径，缩小到 32×32，amber 色填充
- 文字："SudoClaw"，font-heading weight-900
- 用于 Hero 深色背景时：图标 amber + 文字白色
- 用于滚动后浅色背景时：图标 amber + 文字深色
- 实现为内联 SVG + 文字 span，不需要单独图片文件

### 2. Hero 背景纹理

基于 Logo 立方体元素的几何 pattern：
- 用 Logo 的立方体形状做 repeating pattern，低透明度（5-8%）
- 深色背景上的浅色线条感
- 实现为 CSS `background-image` 内联 SVG 或 `<pattern>` 元素
- 或者：用 Logo 路径做一个放大的、模糊的装饰元素放在 Hero 右下角，opacity 10%

## 技术实现

所有 SVG 图标建议用以下方式之一输出：

**方式 A：内联 SVG（推荐）**
直接写在 HTML 中，最灵活，可用 CSS 控制颜色。

**方式 B：SVG sprite**
所有图标合并为一个 `icons.svg` sprite 文件，通过 `<use>` 引用：
```html
<svg><use href="assets/icons.svg#icon-shield"></use></svg>
```

**方式 C：CSS background**
用于背景纹理等装饰性元素。

建议主要图标用方式 A（内联），背景纹理用方式 C。

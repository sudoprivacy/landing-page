# T5 — 场景案例区块（核心重点）

> **依赖：** T0（设计规范）
> **产出：** `<section id="scenarios">` 完整 HTML + JS 交互逻辑
> **内容来源：** `SudoClaw-宣传资料.md` 第 88-212 行
> **复杂度：** 高（10 分类，34 张卡片，Tab 切换 + 滚动交互）

---

## 目标

这是全站内容最重的区块。30+ 个真实业务场景的微案例，需要通过 Tab 切换 + 卡片展示的方式组织，不能让人觉得信息过载。让不同行业的访客能快速找到与自己相关的场景。

## 设计


## JS 交互逻辑

```javascript
// Tab 切换
document.querySelectorAll('.tab-btn').forEach(btn => {
  btn.addEventListener('click', () => {
    // 1. 移除所有 tab active 状态
    document.querySelectorAll('.tab-btn').forEach(b => b.dataset.active = 'false');
    // 2. 设置当前 tab active
    btn.dataset.active = 'true';
    // 3. 隐藏所有 panel
    document.querySelectorAll('.tab-panel').forEach(p => p.classList.add('hidden'));
    // 4. 显示对应 panel
    document.querySelector(`.tab-panel[data-tab="${btn.dataset.tab}"]`).classList.remove('hidden');
  });
});
```

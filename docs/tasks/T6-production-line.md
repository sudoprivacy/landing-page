## 目标

传达两个核心理念：
1. **平台化 = 边际成本递减**——不是做几个机器人，是建一条产线
2. **信任是渐进的**——不要求企业一步到位把决策交给 AI

## 设计

### 背景
深色 `bg-dark` (`#1a1a1a`)，与 Hero 呼应，制造页面节奏变化（浅-深-浅-深）。所有文字为白色/白色半透明。

### 区块标题
"不是做几个机器人，是建一条硅基员工产线" — `text-3xl md:text-4xl font-bold text-white text-center`

### 上半部分：对比表格

用卡片化的对比表（不是传统 HTML table），两列：

```
┌──────────────────────┬──────────────────────┐
│    传统方式           │  SudoClaw 平台化      │
├──────────────────────┼──────────────────────┤
│ 首批上线             │ 平台搭建约 1 周，     │
│ 每个 4-6 周           │ 之后每个 Agent ≈1 周  │
├──────────────────────┼──────────────────────┤
│ 后续扩展             │ 只需写 Skill +       │
│ 每个从零开始          │ Agent 编排            │
├──────────────────────┼──────────────────────┤
│ 系统对接             │ Extension 装一次，    │
│ 每个项目重复对接       │ 所有 Agent 共享       │
├──────────────────────┼──────────────────────┤
│ 运行成本             │ SudoRouter 智能路由，  │
│ 全用高端模型          │ 大幅降低模型成本       │
└──────────────────────┴──────────────────────┘
```

实现方式：
- 外层 `bg-white/5 rounded-2xl p-6 md:p-8 border border-white/10`
- 两列用 grid，左列 `text-white/60`（灰色，表示旧方式），右列 `text-white font-medium`（白色，表示新方式）
- 每行间用 `border-b border-white/10` 分隔

**底部金句引用：**
> 通用大模型是"实习生"——什么都能聊但不懂你的业务。SudoClaw 把规则、流程、知识封装进去，变成"老员工"。

用 `blockquote` + 左侧 amber 色边条：`border-l-4 border-accent pl-6 text-white/80 italic text-lg`

### 下半部分：渐进信任时间轴

标题："信任是渐进建立的" — `text-2xl font-bold text-white text-center mt-16 mb-12`

**桌面端：水平时间轴**
```
  ①──────────②──────────③──────────④
  人工确认     半自动       全面自动     专属模型
```

**移动端：垂直时间轴**
```
  ①  人工确认期
  │
  ②  半自动期
  │
  ③  全面自动化
  │
  ④  专属模型
```

4 个阶段节点：

| # | 阶段名 | 描述 |
|---|---|---|
| 1 | 人工确认期 | AI 只做提示和建议，所有动作人工确认，每周复盘准确率 |
| 2 | 半自动期 | 低风险动作自动执行，高风险仍保留人工确认 |
| 3 | 全面自动化 | 仅决策红线保留人工介入 |
| 4 | 企业专属模型 | 持续积累标注数据触发微调，产出别人买不到也抄不走的核心资产 |

实现方式：
- 水平线：`bg-gradient-to-r from-white/20 via-accent to-accent` （从浅到深，表示信任递增）
- 每个节点：圆形 `w-10 h-10 rounded-full bg-accent text-dark font-bold flex items-center justify-center`
- 节点下方：阶段名（白色 bold）+ 描述（白色半透明 text-sm）
- 用 CSS `@media` 切换水平/垂直布局

**底部安全声明：**
`text-white/60 text-center text-sm mt-12 max-w-2xl mx-auto`
"在整个过程中，您的数据始终在自己的服务器上，AI 的每一步操作都在受控沙箱中完成，跨机构协作通过隐私计算实现"可用不可见"。掌控权从第一天到最后一天，都在您手里。"

### 动效
- 对比表格 `fade-up`
- 时间轴节点逐个淡入，stagger 0.2s
- 渐变线从左到右延展（用 CSS animation `width: 0 → 100%`）

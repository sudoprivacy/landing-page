# 10 — 场景插画：研发与技术

> **用途：** 场景案例区块，"研发与技术" 分类的配图
> **尺寸：** 800×600
> **文件名：** scenario-dev.png

## Prompt

Minimalist isometric illustration of AI assisting software development workflows. Scene elements:

- A code editor window with syntax-highlighted code lines (abstract, not real code — just colored blocks suggesting code structure)
- An amber AI agent reviewing the code with a magnifying glass, marking suggestions with teal annotations
- A CI/CD pipeline visualization: connected circles showing Build → Test → Deploy stages
- An alert/monitoring dashboard in the corner showing server health indicators with green/amber/red status
- A checklist floating nearby with some items checked (release checklist)

Color palette: Amber (#F59E0B) for AI agents and code highlights, teal (#0D9488) for success states, warm cream (#FAF9F0) background. Dark elements for the code editor (dark mode aesthetic for the editor window only). Flat geometric style. The image should feel like a developer's intelligent workspace. No realistic people.

## 用途说明
代表"AI 代码审查、智能告警、发版检查"的研发场景。

# 02 — Hero 产品界面 Mockup

> **用途：** Hero 区右侧的产品截图/界面展示
> **尺寸：** 1200×800
> **文件名：** hero-mockup.png

## Prompt

A clean, modern software application interface mockup on a dark background. The interface shows an AI copilot workspace:

Top: A dark title bar with three window control dots (red/yellow/green) and the text "SudoClaw Copilot" in the center.

Main area (white/light gray background):
- Left side: A chat-like interface where a user message says "帮我准备下周给客户的方案" (in Chinese) in a gray bubble.
- Below the message: An AI response area showing a task breakdown with 4 parallel worker cards arranged in a 2×2 grid:
  - Card 1: "数据采集" with a teal progress bar at 100%
  - Card 2: "痛点提炼" with an amber progress bar at 85%
  - Card 3: "方案设计" with an amber progress bar at 60%
  - Card 4: "ROI 测算" with an amber progress bar at 40%
- Each card has a small gear icon and the label "Worker" with a number.
- Bottom: A status line showing "4 Workers 并行执行中..." with a subtle animation indicator.

Style: Flat design, clean typography (Chinese text), rounded corners on cards, subtle shadows. Colors: white background, amber (#F59E0B) and teal (#0D9488) accents, dark gray text. The overall feel is polished enterprise software, not a consumer chat app. Perspective: slight 3D tilt (5-10 degrees) to add depth, floating on the dark background with a subtle reflection below.

## 用途说明
这是 Hero 区最重要的视觉元素，展示 Copilot + Worker 的实际工作界面。需要让人一看就理解"这是一个任务管理/AI 协作工具，不是聊天框"。

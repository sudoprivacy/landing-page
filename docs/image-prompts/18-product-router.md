# 18 — 产品矩阵插画：SudoRouter 路由图

> **用途：** 产品矩阵区块，SudoRouter 卡片的配图
> **尺寸：** 800×600
> **文件名：** product-router.png

## Prompt

A clean diagram illustrating an AI model router on a warm cream background (#FAF9F0). The concept: one task comes in, the router intelligently selects the best AI model to handle it.

Center: A hexagonal router hub in amber (#F59E0B) with the label concept "SudoRouter" — it has a decision/brain icon inside.

Left side (input): Three types of tasks flowing in, represented by different sized/colored circles:
- Small circle (teal, labeled "L1"): lightweight tasks, largest quantity
- Medium circle (amber, labeled "L2"): standard tasks
- Large circle (dark, labeled "L3"): complex reasoning tasks, smallest quantity

Right side (output): Multiple model provider nodes arranged in a fan pattern, each represented by a simple rounded rectangle:
- Multiple nodes suggesting 20+ providers (some prominent, some faded to suggest many more)
- Connection lines from the router to specific models, with thicker lines to commonly-used ones

The routing paths should visually show: small tasks go to efficient/cheap models, large tasks go to powerful models. A small cost indicator (trending down arrow with ¥ symbol) near the bottom suggests cost optimization.

Style: Flat, geometric, diagram-like. Not a network visualization — more of a clean infographic. Professional and easy to understand at a glance.

## 用途说明
展示 SudoRouter 的核心价值：一个入口、20+ 模型、按任务智能分配，大幅降低成本。

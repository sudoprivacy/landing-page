# 01 — Hero 背景图

> **用途：** Landing Page 首屏背景（深色区块）
> **尺寸：** 1920×1080
> **文件名：** hero-bg.png

## Prompt

Abstract geometric background on a very dark charcoal (#1a1a1a) canvas. Subtle isometric hexagonal cube wireframes scattered across the composition, rendered in amber (#F59E0B) at 8-12% opacity. The cubes reference a blockchain/data-structure visual language — some connected by thin lines suggesting a network graph. A soft radial gradient from the center (slightly lighter, ~#222) fading to edges (#1a1a1a) gives depth. A few cubes near the center are slightly brighter (15-20% opacity) to create a focal point. No text, no logos. Minimal, elegant, premium tech aesthetic. Think Palantir meets Anthropic in color warmth.

## 用途说明
这是 Hero section 的背景底图，上面会叠加白色大标题、CTA 按钮和产品截图。所以画面要足够暗且不抢注意力，几何元素只是氛围。

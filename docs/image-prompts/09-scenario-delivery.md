# 09 — 场景插画：专业服务与交付

> **用途：** 场景案例区块，"专业服务与交付" 分类的配图
> **尺寸：** 800×600
> **文件名：** scenario-delivery.png

## Prompt

Minimalist isometric illustration showing "one human + five AI workers" as a service delivery team. Scene elements:

- Center: One person silhouette (simple geometric, slightly larger) sitting at a desk, representing the senior consultant
- Surrounding the person: Five amber hexagonal cube AI agents arranged in a semi-circle, each with a different tool/function icon above them:
  - Agent 1: Calendar/timeline (project management)
  - Agent 2: Database/pipe (ETL/data cleaning)
  - Agent 3: Chart/graph (analytics & reporting)
  - Agent 4: Neural network (model training)
  - Agent 5: Chat bubble (client communication)
- Thin teal connection lines from each agent back to the central person
- In the background: multiple client icons (smaller, grayed out) suggesting the same team serves many clients simultaneously
- A multiplier indicator "×3" or "×4" near the client area suggesting capacity multiplication

Color palette: Amber (#F59E0B) for AI agents, teal (#0D9488) for connections, dark gray for the human figure, warm cream (#FAF9F0) background. Flat geometric style. The image should instantly communicate "one person empowered by multiple AI specialists = team multiplication."

## 用途说明
代表"1 个顾问 + 5 个硅基员工 = 一个 squad"的服务交付新模式。这是 SudoClaw 最有说服力的场景之一。

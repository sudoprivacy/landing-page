# 14 — 硅基员工产线概念图

> **用途：** "不是做几个机器人，是建一条硅基员工产线" 区块的核心视觉
> **尺寸：** 1600×700
> **文件名：** production-line.png

## Prompt

A wide panoramic illustration of an abstract "AI worker production line" on a dark background (#1a1a1a). The concept: a factory/assembly line metaphor, but for AI agents instead of physical products.

From left to right:
1. **Platform foundation**: A large horizontal base layer labeled elements suggesting infrastructure — servers, databases, API connectors (geometric shapes, teal outlines)
2. **Assembly stations**: Three vertical stages on top of the platform:
   - Stage 1: "Skill" — showing knowledge blocks/modules being loaded (book/document icons dropping into a container)
   - Stage 2: "Agent" — the skill blocks being assembled into a complete AI worker (an amber hexagonal cube taking shape)
   - Stage 3: "Deploy" — the completed AI worker emerging, ready for work (a glowing amber cube with a checkmark)
3. **Output**: Multiple identical AI workers rolling off the line to the right, each going to a different department icon (sales, finance, service, etc.)
4. **Key visual**: A cost curve below the line showing costs decreasing with each additional agent (a declining curve from left to right)

The feeling should be: "the first agent builds the factory, every subsequent agent is cheaper and faster." Warm amber (#F59E0B) dominates the AI agents and highlights. Teal (#0D9488) for the infrastructure layer. White/light text and lines on the dark background. Geometric, clean, no photorealistic elements.

## 用途说明
这是全站最关键的概念图之一——传达"平台化 = 边际成本递减"的核心理念。硅基员工不是一个个手工打造的，而是从一条产线上批量产出。

# 视觉素材统一风格指南

## 所有图片必须遵守的全局约束

### 风格
- **风格关键词：** 扁平化、几何、极简、企业级、科技感但温暖
- **不要：** 3D 写实渲染、赛博朋克、过度发光/霓虹、卡通、stockphoto 风格的人物
- **参考方向：** Palantir 官网的冷静克制 + Anthropic 官网的暖色质感 + Linear/Notion 的极简插画风

### 配色
- 主色：琥珀/暖橙 `#F59E0B`
- 辅助色：青绿 `#0D9488`
- 背景：深色场景用 `#1a1a1a`，浅色场景用 `#FAF9F0`（米白）
- 画面中不出现蓝紫渐变（避免和国内 AI 网站撞车）

### 核心视觉元素
- SudoClaw 的 Logo 是一个几何立方体/六边形（由三角面和菱形面组成的等轴测立方体）
- 这个几何语言可以作为 motif 贯穿所有素材：六边形网格、等轴测透视、几何面片拼接

### 画面尺寸
- 除非单独标注，所有图默认 **1600×900**（16:9 横版）
- 用于网站展示，需确保在缩小到 800×450 时仍清晰可读
